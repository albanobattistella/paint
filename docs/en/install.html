Installing Paint

You can download the software packages arranged for installing on Fedora, Ubuntu, and Raspberry Pi OS from the Releases page.

Building Sources

To build and install "Paint" manually, run the following commands:

git clone https://github.com/esrille/paint.git
./autogen.sh
make
sudo make install
